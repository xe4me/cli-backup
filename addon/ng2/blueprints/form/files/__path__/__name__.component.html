Load dynamic form definition

Bellow is an example of how we can load a chunk of form definition dynamically and then the Fully Distinguished Name will be created on the fly for the child blocks.

Blocks to be loaded

{{ childBlocks | json }}

You can see that this small form definition only has one child block .
So the block-loader will only load that block down bellow himself.

How it works ?

  1. You pass in a chunk of definition
  2. It will load all the blocks that are in the blocks array of that definition down bellow himself
  3. If you want , you can pass in what ever input your desired component/blocks need , which in this case , I'm passing in a control and some labels and button .
