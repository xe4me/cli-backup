AMP's grids use fractional increments. Take the following example which has 2 columns. Both columns stretch equal parts of the 12 column grid. To acheive this is as simple as applying a fraction to the element. In this case we are using: 

                    
                        <div class="grid__item_floated 6/12">

put simply, a column that spans 6 out of 12.

Building on the above example, we will adjust the grid to make it responsive. The AMP css framework consists of multiple responsive breakpoints. In this example, we are changing the structure of our layout, based on if its a tablet device or if its a laptop. To achieve this, we can use syntax like tablet-and-down-6/12 or lap-and-up-1/1. 

                    
                        <div class="grid__item_floated tablet-and-down-6/12 lap-and-up-1/1>
