AMP's css framework has multiple breakpoints. Knowing these will assist you in using it effectively.

Below is a list of breakpoints that our framework supports

Examples of allowed suffixes can be found under the "grid" section.

  • "palm" | "screen and (max-width: 480px)"
  • Use: palm-what ever suffix you need to use
  • "tablet" | "screen and (min-width: 480.01px) and (max-width: 768px)"
  • Use: tablet-what ever suffix you need to use
  • "tablet-and-up" | "screen and (min-width: 480.01px)"
  • Use: tablet-and-up-what ever suffix you need to use
  • "screen-sm-min" | "screen and (min-width: 640px)"
  • Use: screen-sm-min-what ever suffix you need to use
  • "sm" | "screen and (min-width: 640px)"
  • Use: sm-what ever suffix you need to use
  • "tablet-and-down" | "screen and (max-width: 768px)"
  • Use: tablet-and-down-what ever suffix you need to use
  • "lap" | "screen and (min-width: 768.01px) and (max-width: 1024px)"
  • Use: lap-what ever suffix you need to use
  • "lap-and-up" | "screen and (min-width: 769px)"
  • Use: lap-and-up-what ever suffix you need to use
  • "md" | "screen and (min-width: 768px)"
  • Use: md-what ever suffix you need to use
  • "lap-and-down" | "screen and (max-width: 1024px)"
  • Use: lap-and-down-what ever suffix you need to use
  • "lap-and-down" | "screen and (max-width: 1024px)"
  • Use: lap-and-down-what ever suffix you need to use
  • "portable" | "screen and (max-width: 1024px)"
  • Use: portable-what ever suffix you need to use
  • "desk" | "screen and (min-width: 1024.01px)"
  • Use: desk-what ever suffix you need to use
  • "custom-phablet" "screen and (max-width: 540px)"
  • Use: custom-phablet-what ever suffix you need to use
  • "retina" "(-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi)"
  • Use: retina-what ever suffix you need to use
