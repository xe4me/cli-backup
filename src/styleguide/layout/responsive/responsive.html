AMP's css framework has multiple breakpoints. Knowing these will assist you in using it effectively.

Below is a list of breakpoints that our framework supports

Examples of allowed suffixes:

  • "palm" | "screen and (max-width: 480px)"
  • Use: palm-what ever suffix you need to use
  • "tablet" | "screen and (min-width: 480.01px) and (max-width: 768px)"
  • Use: tablet-what ever suffix you need to use
  • "tablet-and-up" | "screen and (min-width: 480.01px)"
  • Use: tablet-and-up-what ever suffix you need to use
  • "screen-sm-min" | "screen and (min-width: 640px)"
  • Use: screen-sm-min-what ever suffix you need to use
  • "sm" | "screen and (min-width: 640px)"
  • Use: sm-what ever suffix you need to use
