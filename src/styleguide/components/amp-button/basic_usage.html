Amp Button

Available attributes
  • [class] = Type of button you would like
  • [data-automation-id] = Important, this is mandatory
  • [attr.theme] = Used for themes

Primary buttons

btn-ok
Ok
btn-ok disabled
Ok

Sample implementation below:

                    
                        
                            <amp-button [attr.theme]="tools" [class]="'btn btn-ok'">
                                Ok
                            </amp-button>

btn-change
Change

Secondary buttons

btn-secondary
Save for later

Sample implementation below:

                    
                        
                            <amp-button [attr.theme]="tools" [class]="'btn btn-secondary'">
                            Save for later
                            </amp-button>
