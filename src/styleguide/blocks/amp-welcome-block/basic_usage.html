Above is an example of a "AMP Welcome" block.

This block is loaded from a form-def.def.json file by the 'amp-block-loader' directive.
It should be the default way of loading it.

Example

In "form-def.def.json" file, add the AMP welcome block like below: 

            
    {
        "name": "Welcome",
        "blockType": "AmpWelcomeBlockComponent",
        "blockLayout": "INLINE",
        "commonBlock": true,
        "path": "modules/amp-welcome-block/components/amp-welcome-block/amp-welcome-block.component",
        "custom": {
            ...
        }
    }

Customisation

In the "custom" section, you have to define:

  • The titles and the background image of the block: 
            
    {
        "custom": {
            "welcomeTitle": "AMP Saver Account",
            "welcomeMessage": "Reach your savings goals sooner",
            "backgroundImageUrl": "url(/assets/images/Intro-background_96DPI.jpg)"
        }
    }
  • The body messages are in point forms. Define the the icon and the text for each point: 
            
    {
        "custom": {
        "welcomePoints": [
           {
            "icon": "time-clock-icon",
            "text": "It will only take about 5 minutes"
            },
           {
            "icon": "eighteen-plus-icon",
            "text": "You must be over 18 years old"
           }
        ]
        }
    }
  • Define the list of actions/buttons. On click of a button, the amp block loader loads the subsequent block. Define the subsequent block that needs to be loaded for each button: 
            
    {
        "custom": {
        "actions": [
            {
                "action": "start",
                "buttonName": "New application",
                "nextBlock": {
                    "name": "contactDetails",
                    "blockType": "AmpContactDetailsBlockComponent",
                    "blockLayout": "INLINE",
                    "commonBlock": true,
                    "path": "modules/amp-contact-details-block/components/amp-contact-details-block/amp-contact-details-block.component",
                    "custom": {
                        "blockTitle": "And your contact details...",
                        "blockFdn" : ["PersonalDetailsSection" , "contactDetails"]
                        {
                    }
                }
            },
            {
                "action": "continue",
                "buttonName": "Continue application",
                "nextBlock": {
                    "name": "basicInfo",
                    "blockType": "AmpBasicInfoBlockComponent",
                    "blockLayout": "INLINE",
                    "commonBlock": true,
                    "path": "modules/amp-basic-info-block/components/amp-basic-info-block/amp-basic-info-block.component",
                    "custom": {
                        "blockTitle": "Where did we leave off?",
                        "blockFdn" : ["PersonalDetailsSection" , "basicInfo"]
                        {
                    }
                }
            }
        ]
        }
    }
