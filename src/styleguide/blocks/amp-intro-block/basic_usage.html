Below is a list of classes that can be used to layout features of the block

It's worth noting, that by default, the block is sitting the same width away from the side of the page, as the menu.

  • .intro-logo
  • Use: placement of the logo
  • .intro-list-container
  • Use: this is meant specifically for list values, it's using flexbox
  • .list-item
  • Use: this is the label text
  • .intro-value
  • Use: this is the value of the item
  • .intro-container
  • Use: this is used for containing any other elements besides a list
