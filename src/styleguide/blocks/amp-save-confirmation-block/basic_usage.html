Above is an example of a "Save Confirmation" block.

This block is loaded from a form-def.def.json file by the 'amp-block-loader' directive.
It should be the default way of loading it.

Example

In "form-def.def.json" file, in any array of blocks: 

            
    {
        "name": "saveClose",
        "blockType": "Amp",
        "blockLayout": "INLINE",
        "commonBlock": true,
        "path": "blocks/amp-save-close-block/components/amp-save-confirmation-block/amp-save-confirmation-block.component",
        "custom": {
            ...
        }
    }
