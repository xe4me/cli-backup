Above is an example of a "Green ID" block.

This block is loaded from a form-def.def.json file by the 'amp-block-loader' directive.
It should be the default way of loading it.

Example

In "form-def.def.json" file, in any array of blocks: 

            
    {
        "name": "greenId",
        "blockType": "AmpGreenIdBlockComponent",
        "blockLayout": "INLINE",
        "commonBlock": false,
        "path": "modules/amp-greenid-block/components/amp-greenid-block/amp-greenid-block.component",
        "custom": {
            "blockTitle": "We now need to verify your identity",
            "controls": [
                {
                    "id": "consent"
                }
            ]
    }
