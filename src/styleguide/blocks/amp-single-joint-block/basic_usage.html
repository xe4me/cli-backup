Above is an example of a "Single or Joint" block.

This block is loaded from a form-def.def.json file by the 'amp-block-loader' directive.
It should be the default way of loading it.

Example

In "form-def.def.json" file, in any array of blocks: 

            
    {
        "name": "singleOrJoint",
        "blockType": "AmpSingleJointBlockComponent",
        "blockLayout": "INLINE",
        "commonBlock": true,
        "path": "modules/amp-single-joint-block/components/amp-single-joint-block/amp-single-joint-block.component",
        "custom": {
            ...
        }
    }

add your applicant def json file etc. applicant.json to your initial application form. ts file susch
