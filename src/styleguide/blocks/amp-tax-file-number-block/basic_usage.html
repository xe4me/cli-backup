Above is an example of a "Tax File Number" block.

This block is loaded from a form-def.def.json file by the 'amp-block-loader' directive.
It should be the default way of loading it.

Example

In "form-def.def.json" file, in any array of blocks:

        "name": "taxFileNumber",
        "blockType": "AmpTaxFileNUmberBlockComponent",
        "blockLayout": "INLINE",
        "commonBlock": true,
        "path": "modules/amp-tax-file-number-block/components/amp-tax-file-number-block/amp-tax-file-number-block.component",
        "custom": {
        "blockTitle": "Questions we need to ask...",
        "controls": [
            {
                "id": "hasTaxFileNumber",
                "buttons": [
                    {
                        "id": "Yes",
                        "value": true,
                        "label": "Yes"
                    },
                    {
                        "id": "No",
                        "value": false,
                        "label": "No"

                ]
            },
            {
                "id": "taxFileNumber"
            },
            {
                "id": "noTaxFileNumberReason",
                "options": [

                        "value": "Children",
