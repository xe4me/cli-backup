Above is an example of a "NewOrExistingCustomer" block.

This block is loaded from a form-def.def.json file by the 'amp-block-loader' directive.
It should be the default way of loading it.

Example

In "form-def.def.json" file, in any array of blocks: 

            
    {
        "name": "newOrExistingCustomer",
        "blockType": "AmpNewOrExistingCustomerComponent",
        "blockLayout": "INLINE",
        "commonBlock": true,
        "path": "modules/amp-new-or-existing-customer-block/components/amp-new-or-existing-customer-block/amp-new-or-existing-customer-block.component",
        "custom": {
            "isInitiallyActive": true,
            "blockTitle": "Tell us whether you're new to AMP",
            "newButtonLabel": "New customer",
            "existingButtonLabel": "Current customer",
            "controls": [
                {
                    "id": "ampNewOrExistingCustomer"
                }
            ]
        }
     }
